The mathematical scope remains narrow:

- finite positive weighted two-way tables;
- weighted Hilbert space projections;
- centered row and column main-effect subspaces;
- order defects of sequential residual stripping under non-product weights;
- one exact rational 2x2 certificate with squared weighted norm `61/177408`.

Release boundary:

- no MaoField empirical positive result;
- no full-panel execution;
- no checkpoint inference;
- no model training;
- no new optimization loss;
- no observed deployed-model residual, interaction, transport or holonomy field;
- no broad new ANOVA, dependent-input or projection-product theory;
- no claim that JSON floats or the deterministic harness prove the theorem;
- duplicate risk remains `MEDIUM`.

Zenodo boundary:

The `.zenodo.json` file is present so that Zenodo can prefer explicit release
metadata over `CITATION.cff` during GitHub release archiving. A DOI exists only
after the repository is enabled in Zenodo and Zenodo successfully archives the
GitHub release.
