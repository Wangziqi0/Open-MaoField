# Paper Source

`preprint.tex` and `preprint.pdf` are the sanitized V2.5 final-synthesis public
draft artifacts for:

```text
Finite Projection Order Defects in Residual Metric Audits:
A Mathematical Starting Point for Black-Box Evaluation
```

The source uses inline `thebibliography` entries. The included PDF was produced
by the external V2.5 synthesis package and verified on node36 by SHA256 and
PDF text/visual checks. Node36 itself did not have a local LaTeX toolchain at
the time of adoption, so subsequent TeX edits must not be made without
recompiling and replacing the PDF.
